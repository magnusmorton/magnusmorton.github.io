DelayRepay

Wed, Jan 20, 2021 We presented DelayRepay: Delayed Execution for Kernel Fusion in Python at the Dynamic Languages Symposium 2020. DLS was supposed to be in Chicago, but alas. The long and short of the paper is that if we trace calls on a NumPy array and delay the execution until needed, we can fuse these calls together when we execute them on a GPU, resulting in improved performance. I’ll include the abstract: Python is a popular, dynamic language for data science and scientific computing.

About
